# Retail BI Forecasting Project

A complete Power BI analytics solution for Retail Sales, Forecasting, Promotions, Inventory Risk, and Store Performance.  
This repository contains the PBIP project, model metadata, DAX scripts, themes, and documentation for enterprise-grade BI development.

---

## 📁 Repository Structure

retail-demand-forecasting/powerbi/
│
├── pbip/                     # Power BI Project (PBIP)
├── model/                    # Model metadata (Tabular Editor export)
│   ├── tables/
│   ├── measures/
│   ├── relationships/
│   └── calculation-groups/
│
├── dax/                      # Individual DAX scripts
├── data/                     
├── theme/                    # Power BI themes
├── docs/                     # Documentation
└── scripts/                 


---

## 🚀 Features

- Full PBIP project under version control  
- Organized measure folders (Revenue, Promotions, Forecast, Inventory, KPI, Store Performance)  
- DAX scripts for code review  
- Ultra Dark Neon Theme v3  
- Forecast diagnostics (Residuals, SHAP, Accuracy)  
- Promotion analytics (Lift, Revenue Split)  
- Inventory risk scoring  
- Store & family performance analytics  
- Clean GitHub-friendly folder structure
